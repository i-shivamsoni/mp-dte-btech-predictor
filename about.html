JEE rank & 12th %

This is an independent, open tool to help Madhya Pradesh B.Tech aspirants understand their MP-DTE counselling options. It is built entirely from official public data: the DTE opening/closing-rank cut-off lists (2017–2025) and the 2026–27 intake (seat-matrix) published on the DTE MP Online portal.

It is a static site — all computation happens in your browser, nothing is uploaded, and no rank or percentage you enter ever leaves your device.

Data coverage

The qualifying-exam (12th-%) merit lists are only published for some years, so the percentage predictor currently covers 2019, 2022-23 and 2024-25. The remaining years are being sourced and will be added once available. The JEE-rank simulator predicts from the most recent rounds (2023–25) — the closest guide to the 2026 cut-offs — while the full 2017–2025 cut-off history powers the Demand and Accuracy pages.

Open source & contributing

This project is open — the full source and the processed (PII-free) data live on GitHub at github.com/i-shivamsoni/mp-dte-btech-predictor. Found a wrong cut-off, a missing college, or a bug? You can open an issue or send a pull request through GitHub — the contributing guide walks through how to do both.
