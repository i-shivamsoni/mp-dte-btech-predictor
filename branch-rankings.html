Demand = how competitive a college’s open/general seat is — lower rank = more sought-after. Ranked per branch from the official 2023–25 cut-offs (no reputation adjustment).

What do Opening / Midpoint / Closing mean? Demand metric chooses the number used to compare colleges: opening rank (best/first admitted — surfaces a college with a strong top but a wide spread), the midpoint of the admitted range (default), or the closing rank (last admitted). Display order then decides whether that list goes most competitive first, least competitive first, or alphabetically. Each row shows all three ranks so you can see the full spread.
