Soon: open source & open for contributions

This site and its underlying data will soon be open-sourced and opened up for community contributions. The goal is simple: a transparent, verifiable tool for Madhya Pradesh B.Tech aspirants, built in the open.

When it lands, you’ll be able to:

  • Read every line — the static site, the deterministic predictor logic, and the data-processing pipeline (no machine learning, no hidden weights).
  • Inspect the data — the official MP-DTE cut-offs (2017–2025) and the 2026–27 seat matrix, exactly as the predictor uses them.
  • Contribute — report issues, fix data-parsing edge cases, add missing years, and improve the methodology via pull requests.

Until then, the predictions remain estimates, not guarantees — always verify on the official DTE portal.
